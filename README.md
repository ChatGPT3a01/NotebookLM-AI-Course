# NotebookLM AI 全能智慧王：突破前瞻未來工作流

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![NotebookLM](https://img.shields.io/badge/NotebookLM-Google-4285F4?logo=google)](https://notebooklm.google.com/)

完整的 NotebookLM 課程教材，涵蓋 19 個單元，從基礎功能到進階自動化系統開發。

---

## 課程特色

- **6 小時完整課程**：從入門到進階，循序漸進
- **HTML 互動簡報**：支援鍵盤導航、全螢幕播放
- **課程認證測驗**：通過測驗即可獲得能力認證證書
- **實戰專案導向**：結合真實應用場景

---

## 課程大綱

### 模組一：NotebookLM 核心功能（單元 1-3）
- 單元 1：NotebookLM 核心功能介紹
- 單元 2：來源管理
- 單元 3：內容生成類型總覽

### 模組二：AI 簡報製作技巧（單元 4-7）
- 單元 4：五大神級簡報風格提示詞
- 單元 5：十五種資料視覺化圖表指令
- 單元 6：簡報模板套用三階段法
- 單元 7：兩次作業法

### 模組三：PDF 編輯工具（單元 8-13）
- 單元 8：為什麼 PDF 無法直接編輯
- 單元 9：PDNob 文字編輯
- 單元 10：NotebookLM 編輯器
- 單元 11：Canva Pro 視覺升級
- 單元 12：Lovart 深度編輯
- 單元 13：實戰應用場景與工具選擇

### 模組四：Chrome 擴充與自動化（單元 14-17）
- 單元 14：Chrome 擴充程式安裝與使用
- 單元 15：NotebookLM 自動化 Web 系統操作
- 單元 16：簡報編修系統 PDF 轉 PowerPoint
- 單元 17：系統部署與遠端存取配置

### 模組五：技術架構深入（單元 18-19）
- 單元 18：前端架構 Vue 3 與 Pinia 狀態管理
- 單元 19：後端架構 Flask 與 Gemini API 整合

---

## 使用方式

### 線上瀏覽
直接開啟 `封面.html` 或任一單元的 `.html` 檔案即可開始學習。

### 鍵盤快捷鍵
| 按鍵 | 功能 |
|------|------|
| `←` / `→` | 上一頁 / 下一頁 |
| `Home` | 跳至第一頁 |
| `End` | 跳至最後一頁 |

---

## 課程認證

完成所有單元後，可在單元 19 最後進行認證測驗：
- 10 題選擇題
- 80 分以上通過
- 通過後可下載 **NotebookLM AI 能力認證證書**
