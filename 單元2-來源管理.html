PDF 文件 - 最可靠的來源格式

推薦優先使用 PDF，品質最穩定！

優點

  • +
    格式固定，不會被網頁雜訊干擾
  • +
    支援掃描版 PDF（OCR 文字辨識）
  • +
    頁碼引用清晰準確
  • +
    離線檔案，內容不會變動

注意事項

  • !
    單檔上限約 500,000 字（約 200 頁）
  • !
    圖片、圖表內的文字可能無法辨識
  • !
    加密或受保護的 PDF 無法處理
  • !
    複雜排版可能影響文字擷取順序
最佳實務：
  • 優先使用「文字版 PDF」而非掃描版
  • 超長文件可拆分成多個來源
  • 確認 PDF 可以選取文字（測試 Ctrl+C 複製）

URL 網頁 - 便利但需注意限制

登入牆問題

需要登入才能看的內容（如付費文章、會員專區）無法擷取

動態載入

JavaScript 動態生成的內容可能無法完整擷取

網頁雜訊

廣告、導航列、側邊欄等非主要內容可能被納入

適合的網頁類型

推薦
  • 維基百科
  • 官方文件、技術文件
  • 新聞媒體文章
  • 部落格文章
  • GitHub README
避免或需確認
  • 需登入的頁面
  • SPA 單頁應用程式
  • PDF 內嵌網頁
  • 圖片為主的頁面
  • 影片串流平台頁面
5 / 20

URL 問題的解決方案

1
登入牆 / 付費牆
先登入後，使用瀏覽器的「另存為 PDF」功能，再上傳 PDF
2
動態載入內容
等頁面完全載入後，選取文字複製，使用「貼上文字」功能
3
網頁雜訊過多
使用瀏覽器的「閱讀模式」後再另存 PDF，或手動複製主要內容
4
確認擷取品質
加入來源後，點擊來源名稱查看預覽，確認內容是否完整
小技巧：Chrome 瀏覽器按 Ctrl + P 可以將任何網頁存成 PDF！

YouTube 影片 - 字幕擷取機制

核心原理：NotebookLM 擷取的是 YouTube 影片的字幕文字，不是影片本身！

字幕來源優先順序

  1. 創作者手動上傳的字幕（品質最高）
  2. YouTube 自動產生字幕（品質次之）
  3. 無字幕（無法處理！）
無法處理的情況：
  • 影片沒有任何字幕
  • 私人或不公開影片
  • 年齡限制影片
  • 直播中（需等直播結束）

品質檢查步驟

1
打開 YouTube 影片
2
點擊 CC 字幕按鈕
3
查看字幕是否準確
4
確認語言是否正確

來源管理技巧：勾選功能

關鍵功能：左側來源列表的勾選框可以控制 AI 參考哪些來源回答！

全選（預設）

AI 會綜合所有來源回答

  • 適合需要全面分析的問題
  • 跨來源比較和整合
  • 尋找共同點或矛盾處

部分勾選

AI 只參考勾選的來源

  • 針對特定文件提問
  • 排除不相關的來源
  • 比較特定來源的差異

實際應用：競品分析

你的筆記本有 A、B、C 三家競品的資料。想知道 A 公司的定價策略時，只勾選 A 公司的來源，避免混淆。想比較三家時，再全部勾選。

來源管理技巧：命名慣例

好的命名讓你一眼就知道來源內容，也讓 AI 引用時更清晰！

依類型分類

[理論] 認知負荷理論.pdf [案例] Tesla 行銷策略分析.pdf [數據] 2026 市場調查報告.pdf [訪談] 客戶A_深度訪談記錄.pdf [競品] Apple_產品規格.pdf

依順序編號

01_研究背景與動機.pdf 02_文獻回顧.pdf 03_研究方法.pdf 04_數據分析.pdf 05_結論與建議.pdf
命名技巧：
  • 使用方括號 [ ] 作為標籤前綴，方便視覺辨識
  • 編號使用兩位數（01, 02...），確保排序正確
  • 避免過長的檔名，保持在 30 字以內
  • 英文或中文皆可，保持一致即可

來源管理技巧：品質把關

檢查項目 高品質來源 需改善的來源
來源可信度 官方文件、學術論文、權威媒體 來路不明的網頁、內容農場
資訊時效性 最新版本、近期發布 過時資訊、已被更新取代
內容完整性 完整文章、全文收錄 擷取不完整、內容截斷
格式可讀性 清晰的文字、良好的結構 OCR 錯誤多、排版混亂
主題相關性 與研究主題直接相關 邊緣相關或完全無關
品質把關清單：加入來源後，點擊來源名稱預覽內容，確認：
  1. 文字是否正確擷取（沒有亂碼）
  2. 內容是否完整（沒有截斷）
  3. 是否為需要的內容（沒有擷取錯誤區域）

常見問題與排解

來源處理失敗

可能原因：

  • 檔案過大超過限制
  • PDF 加密或受保護
  • 網頁需要登入
  • YouTube 沒有字幕

解決方案：拆分檔案、另存 PDF、使用貼上文字

擷取內容不完整

可能原因：

  • 動態載入的網頁
  • PDF 排版複雜
  • 圖片中的文字

解決方案：手動複製貼上、使用其他格式

AI 回答不引用某來源

可能原因：

  • 該來源未被勾選
  • 來源內容與問題不相關
  • 來源擷取品質差

解決方案：確認勾選狀態、檢查來源內容、重新提問

中文 OCR 錯誤多

可能原因：

  • 掃描品質不佳
  • 手寫字體
  • 特殊排版格式

解決方案：使用更高解析度掃描、尋找文字版 PDF
